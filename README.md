## ✨ Overview

InsightFlow is a modern DevOps dashboard designed to give engineering teams a centralized view of deployments, repository health, incidents, AI-powered recommendations, and operational metrics.

The application combines elegant UI design with interactive workflows to simulate a real production engineering platform.

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/anannyami/InsightFlow.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build

```bash
npm run build
```
